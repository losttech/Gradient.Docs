Type MultiRNNCell

Namespace tensorflow.nn.rnn_cell

Parent RNNCell

Interfaces IMultiRNNCell

RNN cell composed sequentially of multiple simple cells.

Example:
Show Example 
num_units = [128, 64]
            cells = [BasicLSTMCell(num_units=n) for n in num_units]
            stacked_rnn_cell = MultiRNNCell(cells)
